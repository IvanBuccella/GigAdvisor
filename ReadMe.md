## Web-App Setup

### First Step

Clone the repository:

```sh
$ git clone https://github.com/IvanBuccella/GigAdvisorAIO
```

### Second Step

Deploy local environment with Docker:

```sh
$ docker-compose up --build
```

### Third Step

Run on your browser

```sh
http://localhost:9000
```
